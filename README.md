# 🔡 Permutation App  
A fast, intuitive tool to generate **N‑letter permutations with fixed positions**, built using **Python + Kivy** and also available as a **GitHub.io web app**.

---

## 🚀 Features

### 🧠 Core Functionality
- Generate permutations of any word  
- Choose desired word length  
- Fix letters at specific positions  
- Fast filtering using custom permutation logic  
- Works offline (APK version)

### 🌐 Web Version Enhancements
- 🌗 Dark/Light theme toggle  
- 📄 Download results as TXT  
- 📱 Mobile‑friendly responsive UI  
- ✨ Smooth animations  
- 🖼️ Logo + branding  
- 📘 Tutorial section  
- ⬇️ GitHub “Download APK” button
